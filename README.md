# Unity templates

A CLI tool for managing custom Unity templates, both project and script

## Installation

```bash
bun install
```

## Development

```bash
bun dev -- [command]
```

## Building

```bash
bun run build
```

## Testing

```bash
bun test
```

## Usage

```bash
unity-templates sync
```

## Commands

- `sync` - Add all the templates to installed Unity versions
